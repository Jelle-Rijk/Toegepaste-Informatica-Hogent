# Wat worden de relaties in eigenschap 4.36 wanneer een binaire hoop wordt opgeslagen in een array met als eerste index 0?

![Eigenschap 4.36](./eigenschap_4_36.png)

```
index = i

left -> i * 2 + 1
right -> i * 2 + 2
parent -> floor((i-1)/2)
```
