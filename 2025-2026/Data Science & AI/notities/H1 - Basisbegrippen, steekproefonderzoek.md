# Lies, damn lies and statistics

Statistieken kunnen misleidend zijn als je een verkeerde plotmethode kiest:

- inconsistente ijking
- data op y-as start niet van nul (vertekend beeld van hoe groot veranderingen zijn)
- verschillende weergave van data (vb. 21% voor / 79% tegen)
