# Measures of central tendency

- Modus -> Element dat het vaakst voorkomt.
- Mediaan -> Middenste element in een gesorteerde lijst.
- Gemiddelde -> Totaal / aantal <br>_(Arithmetic mean = gemiddelde)_

Als je gemiddelde vermeldt ook altijd standaardafwijking vermelden.

μ -> gemiddelde van een populatie <br>
x̄ -> gemiddelde in een sample

# Measures of dispersion

Enkel gebruikt bij kwantitatieve variabelen.

**Percentiel** -> 40e percentiel = 40% van de waarden is kleiner dan deze waarde.

**Kwartielen** -> data wordt opgedeeld in 4 even grote delen, grenzen liggen op:

- minimumkwartiel = 0%
- Q1 - eerste kwartiel = 25e percentiel
- Q2 - tweede kwartiel = 50e percentiel
- Q3 - derde kwartiel = 75e percentiel
- max = 100%

**Bereik** (range) = maximumwaarde - minimumwaarde

**Interkwartielafstand** = |Q3 - Q1| -> dit bevat 50% van de waarden. Is vooral geschikt voor kwantitatieve waarden zonder normale distributie.

_Normale distributie = de waarden zijn symmetrisch rond het gemiddelde verdeeld in een grafiek die op een bel lijkt. Meer info in een verder hoofdstuk._

## Variantie en standaardafwijking

**Variantie** -> Vooral geschik voor kwantitatieve metingen met normale distributie.

Sample van grootte n / Populatie van grootte N:

- Bereken eerst het gemiddelde van de sample.
- Voor elk element -> (element - gemiddelde van sample / populatie)<sup>2</sup>
- Je neemt de som van alle uitkomsten van de vorige stap.
- Deel de uitkomst door n-1 of N (volledige populatie)

> n - 1 bij samples i.p.v. n?
>
> Geeft consistent een accuratere standaardafwijking terug. Waarom dit zo is, valt niet onder deze cursus. <br>
> Let op met Python library defaults, `pandas` gebruikt standaard de sample functie, `numpy` de populatiefunctie.

In libraries is ddof vaak een parameter = Delta Degrees of Freedom (0 = delen door n, 1 = delen door n - 1)

**Standaardafwijking** = vierkantswortel van de variantie.

- De standaardafwijking is altijd positief (want het is een vierkantswortel)
- What would be the smallest possible value for the standard deviation? What would that value indicate about the data?
- What effect do outliers have on the mean and standard deviation?
- What is the unit of measurement for the mean and standard deviation? How does that compare to the unit of measurement for the variance?

## Scheefheid en kurtosis

Positively skewed -> er zijn meer kleine waarden (langere staart rechts) <br> Negatively skewed -> er zijn meer grote waarden (langere staart links)

Kurtosis => geeft aan hoeveel uitschieters er zijn

Skewness en kurtosis moeten dicht bij 0 liggen bij normale distributies.

# Visualisatie (Seaborn)

- Kwalitatieve metingen: bar charts
- Kwantitatieve metingen: histogram, boxplot, density plot

Pie charts zijn niet gemakkelijk om af te lezen -> daarom geen deel van deze cursus.

De meeste data scientists korten seaborn af als `sns`.

| Nederlandse naam | Engelse naam | Seaborn methode                                                 |
| ---------------- | ------------ | --------------------------------------------------------------- |
| Staafdiagram     | Bar chart    | sns.catplot(data=`dataframe`, kind='count', y=`kolomnaam-y-as`) |
| Boxplot          | Box plot     | sns.boxplot(data=`dataframe`, x=`kolomnaam-x-as`)               |

## Boxplot

Toont de interkwartielafstand (Q3 - Q1) = box. De lijn in de box is de mediaan.

De boxplot heeft twee "whiskers". Ze stellen een minimum- en maximumwaarde voor. Data die buiten deze grenzen vallen worden gezien als outliers.

De minimumwaarde wordt gelegd op: Q1 - 1.5 \* interkwartielafstand.

De maximumwaarde wordt gelegd op: Q3 + 1.5 \* kwartielafstand

## Histogram

De data wordt verdeeld in bins van even grote intervallen. De data wordt daarbinnen als een soort bar chart weergegeven.

Verschil met bar chart -> bar chart toont voor elke waarde een aparte balk.

## Density plot

Lijkt op een histogram, maar is een doorlopende lijn zodat je beter de vorm van de belcurve kan volgen.

Een violin plot is de combinatie van een density plot en boxplot.

## Richtlijnen

Bij visualisatie:

- Altijd de assen labelen
- Altijd een titel opgeven
- Geen 3D-plots gebruiken (vervormt de data)
- Toon de volledige as als dit relevant is inclusief de origin. (voorkomt misleidend gebruik van de grafiek)
- Vermeld de eenheid (als er een is)
- In rapporten, papers, theses -> altijd een caption met uitleg over hoe je de grafiek afleest en wat er te zien is.
